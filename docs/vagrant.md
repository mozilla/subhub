# Vagrant

## Setup MacOS
* Install Virtualbox `brew cask install virtualbox`
* Install Vagrant `brew cask install vagrant`

## Running

* Starting: `vagrant up`
* Destroying: `vagrant destroy`
* SSH: `vagrant ssh`
* (Re) Provisioning: `vagrant provision`

## Running Unit Tests

* `cd /opt/subhub`
* `python3 dodo.py`
* `doit test`
